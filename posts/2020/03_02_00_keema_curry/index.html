キーマカレー1.1。

水に関する記述が抜けていたので追記。

急にどうしたお前

材料

  • 玉ねぎ : 2個

  • 人参 : 0.5個

  • ひき肉 : 240g

  • カレールー : ジャワカレーキーマ中辛ひと箱の半分(4皿分)

  • 油 : 適量(うちではオリーブオイル)

  • 水 : 250ml (ルーの箱に書いてあった200mlだと少し足りない)

  • 米

手順

同時にゆで卵などやっておくとよい。

下ごしらえ

  1. 野菜をすべてみじん切り

炒める

  1. 油をひいて着火

  2. 野菜とひき肉を突っ込む

  3. いい感じになるまで炒める(5分程度)

水を足す

沸騰するまでそのまま加熱。

混ぜる

  1. 火を止める

  2. ルーを投下

  3. 溶かして混ぜる

  4. 弱火で5分ほど

盛る

盛る。
